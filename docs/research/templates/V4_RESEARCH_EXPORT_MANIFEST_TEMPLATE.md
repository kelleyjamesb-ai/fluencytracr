## Safety Review

Confirm the export does not contain:

- raw user IDs,
- emails or names,
- stable hashed user IDs,
- raw skill names,
- prompts,
- outputs,
- transcripts,
- action rows,
- raw event rows.

## Small-Cell Review

Confirm no surfaced row has user or cohort count below 5. Low-count rows may be
used only as internal QA evidence that a source is not yet safe to interpret.
